# Bodhitree_2.0


A dynamic website showcasing an e learning platform (Bodhitree 2.0).

# Technologies Used:
- Front end: Html, CSS , along with Bootstrap for designing purpose
- Backend: done in PHP
- Database: MySQL database
- Done using the xampp server
